Installation

Cocoapods Service Provider

CocoaPods

(CocoaPods) is a dependency manager for Cocoa projects. You can install it with the following command:

$ gem install cocoapods

CocoaPods 0.39.0+ is required to build Oatmeal

To integrate Oatmeal into your Xcode project using CocoaPods, specify it in your Podfile:


source 'https://github.com/CocoaPods/Specs.git'
platform :ios, '8.0'
use_frameworks!

pod 'Oatmeal'
Then, run the following command: $ pod install

Sweet! We got the hard part over with. Just kidding, let's see if everything is working. Navigate to your ViewController or AppDelegate and add import Oatmeal to the top of your class and attempt to build. If no compile errors come up, we're good to go.

Create The OatmealService Provider

This provider is required for Oatmeal to work. You can append any classes you want Oatmeal to know about to the provides array.

   
import Foundation  
import Oatmeal  

final class OatmealServiceProvider : ServiceProvider
{
     /*
        This is the default service provider for Oatmeal. 
        Objects removed will not be accessible unless bound later.
     */
    var provides : [Resolveable.Type] = [
        FileCache.self,
        MemoryCache.self,
        Configuration.self,
        FileLog.self,
        Reachability.self,
        Networking.self,
        HttpLog.self,
        Serializer.self,
        Reflections.self,
    ]
    
    func registerCustomTypes() -> [Any.Type]
    {
        return [Any.Type]()
    }
}
